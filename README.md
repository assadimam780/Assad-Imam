what is Terraform ?
Terraform is an open-source tool created by HashiCorp that lets you define and manage infrastructure using code instead of manual setup.
In simple terms, it’s part of a practice called Infrastructure as Code (IaC)—you describe what servers, networks, databases, and cloud resources you want, and Terraform builds and manages them for you.

what is resource group?
A Resource Group is a logical container used to organize and manage related resources in cloud platforms—especially in Microsoft Azure.
A Resource Group is like a folder where you keep all related cloud resources together, such as:
Virtual machines
Storage accounts
Databases
Networks

What is Storage account?
A Storage Account is a cloud service used to store and manage data like files, images, videos, backups, and logs. In Microsoft Azure, it acts as the main container for all your storage services.
Inside a Storage Account, you can store:
Blobs → images, videos, documents
Files → shared file systems
Queues → messages for apps
Tables → structured NoSQL data
